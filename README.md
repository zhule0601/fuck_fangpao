# fuck_fangpao

杭州市余杭区`福鼎家园`放炮记录(阿里西溪园区旁)


### 福鼎家园放炮太过恶劣!!

很多人在阿里西溪园区工作, 为了工作方便都是在周围 1 公里内租的房子, 核心片区就是`福鼎家园`这块. 
但是每天上班就很累了, `福鼎家园`的原住民(~sb~)们天天大早上放炮, 吵得整个小区不得安宁! 根本无法休息, 每声炮声都直击心灵!


### 主要功能
1. 每个人 15 min 可以进行一次放炮记录(ip 识人), 并留下一句备注(~骂人的话~)
2. 生成过去一段时间的放炮记录进行展示
3. 以此图告诫打算租房的同学放弃`福鼎家园`片区. btw: 离得近的`西溪北苑`, `海兴雅苑`, `富力-西溪悦居`也都能听到炮声
